#### Installation
```
$ git clone git@github.com:KevinPy/Iceberg.git
$ cd Iceberg
$ npm install
```
#### Scripts
`npm start` : Launch `webpack-dev-server` with Hot Module Replacement - http://localhost:8080<br />
`npm run build` : Compile **/src** folder and create **/dist** folder<br />
#### Structure
```
/src
    |- /assets
        |- /images
        |- /styles
            |- /main.scss
    |- /js
        |- components
            |- App.jsx
    |- index.html
    |- index.js
```
